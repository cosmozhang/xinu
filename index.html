Installation of Xinu on Raspberry Pi.

Hardware:

  • Raspberry Pi B+ ### System:
  • Embedded Xinu

Procedure

General Procedure

  • git clone https://github.com/xinu-os/xinu
  • cd xinu
  • make -C compile PLATFORM=arm-rpi

Set up a ARM cross-compiler

  • wget ftp://ftp.gnu.org/gnu/binutils/binutils-2.23.tar.gz
  • tar xvf binutils-2.23.tar.gz
  • mkdir binutils-2.23-build
  • cd binutils-2.23-build
  • ../binutils-2.23/configure --prefix=/opt/arm-none-eabi-dev --target=arm-none-eabi --disable-nls

Set up the gcc for arm-none-eabi

  • wget ftp://ftp.gnu.org/gnu/gcc/gcc-4.8.2/gcc-4.8.2.tar.bz2
  • tar xvf gcc-4.8.2.tar.bz2
  • mkdir gcc-4.8.2-build
  • cd gcc-4.8.2-build ** gmp: sudo apt-gei install libgmp3-dev ** mpfr: sudo apt-get install libmpfr-dev libmpfr-doc libmpfr4 libmpfr-dbg ** mpc: apt-get install libmpc
  • ../gcc-4.8.2/configure --prefix=/opt/arm-none-eabi-dev --target=arm-none-eabi --enable-languages=c,c++ --without-headers --disable-nls
  • make all-gcc all-target-libgcc
  • sudo make install-gcc install-target-libgcc
  • export PATH="$PATH:/opt/arm-none-eabi-dev/bin" (in .bashrc)
  • make -C compile PLATFORM=arm-rpi COMPILER_ROOT=/opt/arm-none-eabi-dev/bin/arm-none-eabi-
